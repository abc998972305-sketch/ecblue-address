# ECblue Basic — Адресатор

Утилита для изменения Modbus-адреса вентиляторов **ZIEHL-ABEGG ECblue Basic** через конвертер **MOXA UPort 1150** (USB → RS-485).

## Назначение

Позволяет последовательно адресовать несколько ECblue Basic: подключаешь одно устройство, меняешь адрес, отключаешь — подключаешь следующее. Программа автоматически обнаруживает подключение/отключение и повторно авторизуется.

## Требования

- Windows 7/10/11
- Python 3.8+
- MOXA UPort 1150 с установленным драйвером
- Подключение: RS-485 (A/D+ и B/D−) к модулю AM-MODBUS на ECblue Basic

## Установка

```
pip install pymodbus pyserial
```

## Запуск

```
python ecblue_address.py
```

## Как использовать

1. Подключи ECblue Basic (адрес по умолчанию — **247**) через MOXA к ПК
2. Выбери COM-порт в программе
3. Программа автоматически подключится, введёт PIN и покажет текущий адрес
4. Введи новый адрес (1–246) и нажми **«Записать адрес»**
5. Подтверди — адрес сохранится
6. Отключи устройство, подключи следующее — цикл повторяется

## Modbus-параметры

| Параметр | Значение |
|----------|---------|
| Скорость | 19200 baud |
| Чётность | Even (8E1) |
| Адрес по умолчанию | 247 |
| Регистр адреса | h3 (биты [15:8]) |
| PIN разблокировки | 0010 |
